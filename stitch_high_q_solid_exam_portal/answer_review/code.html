Detailed Explanation

To find the local minimum of the function \( f(x) = x^3 - 3x^2 + 2 \), we must first find its first derivative and set it to zero to identify critical points.

f'(x) = 3x^2 - 6x
Setting f'(x) = 0:
3x(x - 2) = 0
Therefore, x = 0 and x = 2 are critical points.

To determine which point is a local minimum, we use the second derivative test: \( f''(x) = 6x - 6 \).

  • At \( x = 0 \), \( f''(0) = -6 \). Since \( f''(0) < 0 \), there is a local maximum at \( x = 0 \).
  • At \( x = 2 \), \( f''(2) = 12 - 6 = 6 \). Since \( f''(2) > 0 \), there is a local minimum at \( x = 2 \).
